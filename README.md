# OpenShelf

OpenShelf is an open-source macOS shelf utility inspired by temporary drag-and-drop shelf apps.

Drag files to the edge of the screen, temporarily store them in a floating shelf, and drag them out later.

## Features

- Left/right screen-edge trigger
- Floating shelf window
- Drag files into the shelf
- Drag files back out
- Double-click to open files
- Menu bar utility mode

## Build from source

```bash
swift build
swift run
```
